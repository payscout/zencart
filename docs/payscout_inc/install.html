Installation

Download and unzip the payment module package from the Zen Cart website and upload the files to your store. Now go to your admin > modules >payment and you should see an entry for Payscout Inc:

Let's run through the options as shown on the right.

Enable Payscout Payments - fairly simple - True will display Payscout Inc to your customers, false will disable it.

Payment Zone - this is a standard Zen Cart setting and will allow you to restrict Stripe payments to zone(s) that you have setup in your store

Set Order Status - will be attached to successful orders made using Payscout

Transaction Mode - Test will allow you to setup the module to your satisfaction and a marker will appear in the store front end warning that the module is in test mode.
Live mode allow transaction with real cards.

API Username - This wll be same as your payscout merchant username.

API Password - This wll be same as your payscout merchant password.

API Token - these are obtainable through your Payscout Merchant dashboard and should be copied from there and pasted here.

Transaction Mode - Select transaction mode Live/Test.

Authorization Type - Select authorization type debit/credit.

Database Storage - Select accordingly if you want to store transactions logs in database.

Order Status - Select order status after payment success.
